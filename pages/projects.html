Android App project

The idea behind this project is essentially to create an android app based off the ACM website. Wether you're an experienced app developer, taken the android class here or just want to learn the ins and outs of android app development, this work shop is for you!

  • What should I excpect?
    • Members will be given the task to create their own android app version of this ACM Website.
    • Every workshop, there will be a quick lecture to go over basic concepts for app development.
    • Each members app will be earn point based on tasks the project leader have set out.
    • At the end of the semester, the project points will be compared and prizes will given for the best pojects.
  • What do I need to have?
    • Good Understanding of object oriented programing.
    • Familiarity with Extensible Markup Language (XML).
    • Android Studio installed.
    • Git or Git bash installed on your laptop
    • A Git Hub Account with a new clean repository
